# 🩸 Blood Match

> **Descubra a harmonia do seu relacionamento através da tradição e da Inteligência Artificial.**

---

## 📝 Descrição do Projeto

O **Blood Match** é uma aplicação web interativa que combina a milenar tradição japonesa **Ketsueki-gata** (a crença de que o tipo sanguíneo define traços de personalidade e compatibilidade amorosa) com o poder da Inteligência Artificial moderna. 

Através de uma interface de chat fluida e dinâmica, o usuário informa o seu tipo sanguíneo e o da pessoa desejada. O sistema processa esses estados e aciona um agente inteligente focado em análise comportamental para gerar um relatório exclusivo de compatibilidade.

---

## 🛠️ Tecnologias Utilizadas

O projeto possui uma arquitetura simples e dividida em duas partes essenciais:

* **Frontend:** Construído utilizando **Preact**, **TypeScript** e empacotado via **Deno**.
* **Backend:** Desenvolvido em Python com **FastAPI** e **smolagents**, responsável por gerenciar as sessões do chat e consumir a API de um modelo de **LLM** para gerar as análises.
